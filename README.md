# Project Title

Contact App using React Vite

## Features

- Contact List
- Update Contact
- Add a new Contact
- Delete a Contact

## Tech Stack

**Framework:** React

**API:** REST API

## Run Locally

Clone the project

```bash
  git clone https://github.com/sharma03r/contact-app
```

Go to the project directory

```bash
  cd contact-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
